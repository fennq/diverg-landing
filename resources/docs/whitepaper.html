Diverg Whitepaper

Security & Blockchain Investigation — unified platform, fact-only reporting

Introduction

Diverg is a security technology company focused on multimodal investigation and efficiency. We provide a single platform that combines traditional web security assessment with blockchain investigation and open-source intelligence (OSINT). This document describes what Diverg is, our mission and principles, the technology behind the platform, and how we intend to build on it.

What is Diverg?

Diverg is an autonomous security and blockchain investigation platform. It performs reconnaissance, vulnerability scanning, and chain analysis in a unified workflow. Investigations can be run on demand (e.g. from a browser extension or CLI), scheduled for continuous monitoring, or deployed in enterprise environments with custom integrations.

Unlike toolchains that require analysts to switch between many separate products, Diverg is built so that one investigation can span web surfaces, wallets, and public data—with a single report that ties findings together.

Mission

We believe security investigations should be comprehensive, factual, and accessible. Diverg was founded to bridge the gap between traditional web security and blockchain investigation—providing one platform that sees the full picture.

Whether you are tracking an attacker across multiple chains, assessing a token launch for investment, or conducting authorized penetration testing, Diverg is designed to deliver the tools you need with data you can trust.

Technology

Diverg is built on a modular, skill-based architecture powered by modern reasoning systems. Each “skill” corresponds to a well-defined capability (recon, web vulns, headers/SSL, auth testing, API testing, chain analysis, etc.). The system follows established methodologies—such as the OWASP Testing Guide and PTES—while adapting dynamically to the type of target (e.g. web app vs. DeFi protocol vs. wallet).

We integrate with industry-standard data sources (including Solscan Pro, Arkham, Etherscan, Bubblemaps) and maintain the ability to operate in a degraded or standalone mode when API keys are not configured. In those cases, reports clearly indicate which checks were skipped rather than presenting synthetic or placeholder data.

Multi-Surface Investigation

Investigations can span:

  • Web: Port scanning, subdomain enumeration, technology fingerprinting, XSS/SQLi/CSRF and misconfiguration checks, security headers, TLS, auth and API testing.
  • Blockchain: Wallet and token analysis, transfer normalization, flow diagrams, counterparty mapping, and integration with block explorers and labeling services.
  • OSINT: WHOIS, DNS, email patterns, historical data (e.g. Wayback), and correlation across surfaces.

Results are presented in a structured format (findings, severity, evidence) and can include interactive flow visualizations for funds and relationships.

Fact-Only Reporting

We never present placeholder or synthetic data as real. When external APIs are configured, you get live data; when they are not, reports explicitly state what was skipped and why. This transparency is core to how we build: analysts and stakeholders should be able to rely on every line in a Diverg report.

Use Cases

Diverg supports:

  • Crypto & DeFi security: Smart contract and token launch assessment, deployer and holder analysis, rug-pull indicators.
  • Incident response: Attack timeline reconstruction, attacker wallet tracking, exchange deposit identification, evidence packaging.
  • Due diligence: Pre-investment or pre-acquisition security and infrastructure review, on-chain entity checks.
  • Continuous monitoring: Scheduled scans and wallet alerts, with delivery via dashboard or messaging (e.g. Telegram).

Roadmap and Extensibility

We are expanding the documentation with installation guides, configuration reference, and API documentation. The same layout and design used here will be used for all future docs so that the experience stays consistent and professional. Enterprise deployments, custom skills, and deeper integrations are available on request.

Conclusion

Diverg aims to be the platform of choice for teams that need both web security and blockchain investigation in one place, with fact-only outputs and a clear methodology. For a hands-on introduction, request a demo or reach out at hello@diverg.sec.
