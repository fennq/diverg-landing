What is Diverg?

A unified platform for security and blockchain investigation, web, chain, and OSINT in one place.

Overview

Diverg is an autonomous security and blockchain investigation platform. Unlike toolchains that force analysts to juggle block explorers, web scanners, and OSINT tools separately, Diverg provides a single environment where one investigation can span web infrastructure, wallets and tokens, and open-source intelligence, with one report that ties everything together.

Key points

  • Unified: Web security, blockchain analysis, and OSINT in one workflow
  • Fact-only: We never show placeholder or synthetic data; reports clearly state what was verified and what was skipped
  • Modular: Skill-based architecture that adapts to the target (e.g. DeFi vs. traditional web)
  • Multi-modal: Dashboard, hosted API, open-source CLI (diverg-auto), and Chrome extension (source)

Diverg Open vs private

Diverg Open is self-serve: console, API, extension, and the open-source scanner for authorized use. Private engagements are investigations and security assessments delivered with our internal tooling, deeper and scoped to your needs. Book private work: company@divergsec.com.

The problem we solve

Security investigations today require switching between many disconnected tools: block explorers, labeling services, vulnerability scanners, DNS/WHOIS tools, and manual correlation in spreadsheets. That fragmentation wastes time and increases the risk of missing connections. At the same time, many tools present synthetic or example data when real data is unavailable, which undermines trust in findings.

The Diverg approach

Diverg unifies reconnaissance, vulnerability assessment, and chain analysis into a single platform. You submit a target (URL, domain, or wallet); the system runs the right combination of skills and correlates results. Reports distinguish verified findings from skipped checks, so you always know what you can rely on. For the full technical and product story, see the Whitepaper.
